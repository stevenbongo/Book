Chapter 1 Introduction

본 글은 A2019 Package인 Web-Automation에 대해 설명하기 위해 작성하였다.
해당 package는 A2019 Web-Automation에서 무료로 다운받을 수 있다.

1.1 Overview

  • Web-Automation은 Chrome browser에서 JavaScript를 활용하여 업무를 자동화하기 위해 사용되는 package이다.
  • Web-Automation은 Chromedriver와 Selenium을 기반으로 작동한다.
